# Pinball - Polymatic Example

Pinball game with editable SVG table. Implemented using [Polymatic](https://github.com/piqnt/polymatic), [Planck.js](https://github.com/piqnt/planck.js) and [svg-play](https://github.com/shakiba/svg-play).

[Play Live Demo](https://piqnt.github.io/polymatic-example-pinball/)

### How to run the code

To run or build the source code in this repository you need to have node.js/npm installed.

Install this project dependencies:

```sh
npm install
```

To run the project locally:

```sh
npm run dev
```

This will print out the url where you can open the project.

To build the project for production:

```sh
npm run build
```
